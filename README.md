# Learning Project
## Description
A learning project using Flask, SQLite, scikit-learn, and Plotly/Dash.
## Acceptance Criteria
* The application must have a home page that displays a message.
* The application must have a data page that displays a line chart.
* The application must have a model that can be trained and make predictions.
* The application must have a database that can store and retrieve data.
## Setup Instructions
1. Clone the repository.
2. Install the required dependencies using pip.
3. Run the application using Flask.
## Example Use Cases
* Train a model using the `train_model` function.
* Make predictions using the trained model.
* Display a line chart using Plotly/Dash.
